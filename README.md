# MatrixGeneration

## Overview, Purpose & Features
- it is a simple app. It will generate matrix in recyclerview, also add timer to each view and user can start and stop timer.

## Concepts Used
- Navigation Component
- Recycler View
- Fragments
- ViewModel && LiveData
- ViewBinding
